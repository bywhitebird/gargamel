# Introduction

## Overview of Kazam

Kazam is a versatile library that empowers developers to effortlessly generate components in a wide range of formats and tools, including React, Vue, Svelte, and Figma, all from a single codebase and unified tool, such as Kaz or Figma.

## Benefits of using Kazam

Kazam allows you and your team to have a single source of truth for your components, reducing duplication and ensuring consistency across your projects. With Kazam, you can create and maintain a library of reusable components that can be easily integrated into different projects, regardless of the underlying technology stack.

## Use case examples

- Create a component library for your company's design system
- Create an open source component library for the community
